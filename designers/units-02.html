Pixels.

A "px" is a unit of measurement in CSS that is commonly read as "pixel" or "screen pixel" in an absolute. There has also been a false notion that the PX unit is a non-linear unit of measurement.

The size of a px is defined by the browser by a establishing an anchor pixel. This anchor pixel sets the stardard for all other pixels on the page. So when you crack open Photoshop highlight some box and confidently announce that a box should be 30 pixels wide, what you are really saying is "the box should be 30 of my computers pixels wide on every computer there is". Definately not an easy task.

Px works this way due to variations in screen resolution. For instance a pixel on a Retina Display iPhone is 1/4th the size of a pixel on a non retina iPhone, so the phone's software sets an ancor pixel to perserve consistency. It's all pretty boring stuff.

Also, px can only be expressed in whole units, so there is no such thing as a half pixel.
