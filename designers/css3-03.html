Drop Shadows

Drop Shadows come it two forms, box-shadows and text-shadows

.demo001{
background-color: blue;
opacity:0.3;
}

<div class="demo001"some stuff></div>
some stuff

Once something has opacity, all it's contents are also opaque. This is handy in animations especially and can be used to accomplish native CSS fades. Alpha transparent backgrounds are better accomplished like this:

.demo002{
background-color: rgba(17,142,200,0.5);
opacity:0.3;
color:#000;
}

<div class="demo002"some stuff></div>
some stuff

The differense here is the use of alpha transparency in the background-color, which isn't supported in IE8.

Another important distinction is that rgba and opacity use alpha transparency, or "blend mode normal" in Photoshop terms.
