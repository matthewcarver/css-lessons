Part 2: How CSS changes a page.

CSS turns the plain, milquetoast browser defaults into something with a little more class and eventually using CSS you can turn a page into something like this. CSS in this example:

html{
background:#b4d7e7;
}

body{
width:50%;
margin:0 auto;
font-family:Helvetica, sans-serif;
text-shadow:0 1px 0 #fff;
}

h1{
color:#333;
}

p{
font-weight: bold;
font-size:20px;
}

a{
color:#666;
}
