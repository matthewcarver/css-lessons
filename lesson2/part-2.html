Part 2: The Box Model

Every element on ever webpage thats ever been built it made of rectangles (and squares if you really want to get nit picky). I know you think you've seen circles in web pages, but you haven't. Those are rectangles that a developer made look like circles.

This is something difficult to explain, but it goes something like this:

Lets play with the box model a little and start to get a feel for it. We'll start with our example from the previous chapter.


<div style="background:blue"> some stuff </div>
some stuff

Lets set some sizing properties.







Width & Height:

<div style="background:blue; width:200px; height:200px;"> some stuff </div>
some stuff






Padding:

<div style="background:blue; width:200px; height:200px; padding:20px;"> some stuff </div>
some stuff






Border

<div style="background:blue; width:200px; height:200px; padding:20px; border:5px solid red;"> some stuff </div>
some stuff






Margin:

<div style="background:blue; width:200px; height:200px; border:5px solid red; padding:20px; margin:40px;"> some stuff </div>
some stuff

Notice how the padding moved the text within the blue box, but the margin moved the blue box?
