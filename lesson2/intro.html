Intro: Regarding Objects

Webpages are composed of Objects. In HTML an object looks like this:

<div> some stuff </div>

or this:

<h1> Header </h1>

or this:

<p> Lorem ipsum dolor sit amet, consectetur adipisicing elit, sed do eiusmod </p>

The point is that there are tons of objects, each with it's own purpose, but this is a CSS lesson so we'll assume youre familiar with HTML enough to nod your head and go along from here.
