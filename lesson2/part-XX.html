Part 2: Making Objects Look (Less) Like Pure Shit

Adding a background color to a div is a beginning, but there are a myriad of rules an object can be given and an infinite way to apply rules to objects to produce a website.

Inline styles is one way, but adding a class or an ID is a better way

To add a class to an object you simply add the attribute of "class", like this:

<div class="some-class"> some stuff </div>

Classes are identifiers for CSS to hook into. It draws a connection between CSS and an object. It's like the hookshot in Legend of Zelda, but with code.
