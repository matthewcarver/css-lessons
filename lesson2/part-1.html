Part 1: Making Objects That Look Like Rubbish

The first and most direct way to give and object a style is inline. To write inline styles you add style="" to the object. So this:

<div> some stuff </div>

Which looks like this in the browser

some stuff

Becomes:

<div style="background:blue"> some stuff </div>

Which looks like this in the browser

some stuff

Let's make this more useful

There is a huge amount of things you can control with CSS, too many to list here but there are plenty of cheat sheets online. Rules are applied in a specific format (also called a syntax). The syntax for CSS is [property]:[rule];.

"Property" is the way in which an object is affected

"Rule" is how the property is applied

We're going use a few practical examples of some rules and their affects.







BACKGROUND COLOR

Description: "background-color" adjusts the color of the object. Colors can be set using hex values or

<div style="background:blue;"> some stuff </div>

Produces

some stuff






COLOR

Description: "color" adjusts the color of the text within an object.

<div style="background:blue; color:red;"> some stuff </div>

Produces

some stuff






FONT-SIZE

Description: "font-size" adjusts the size of the text within an object.

<div style="background:blue; color:red; font-size:40px;"> some stuff </div>

Produces

some stuff
